SafariHere

Intelligently opens a new Safari window or tab.

If you work with multiple spaces, and have Safariset to open links in new tabs (not new windows), the resulting behaviour can reduce your life expentancy.

[video]

Set SafariHere as the default browser instead of Safari to make things behave a bit better when you click on links.

SafariHere will create a new Safari window in the current space if necessary, and open the link there. No more flying around between spaces.

Free.

Works well with [WebBuddy].
